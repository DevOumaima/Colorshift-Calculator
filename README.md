# Colorshift-Calculator
This is a fully functional, interactive calculator built as a web application using HTML, CSS, and JavaScript. It provides a clean, user-friendly interface for performing basic arithmetic operations, with an added twist of customizable color themes to enhance the user experience. The project demonstrates fundamental web development skills, including DOM manipulation, event handling, and responsive design.
